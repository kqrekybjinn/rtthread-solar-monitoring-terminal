# License Options

当前仓库已添加 `LICENSE`，采用 `CERN-OHL-P-2.0` 发布硬件资料。

## 硬件许可证建议

| 许可证 | 适用情况 |
| --- | --- |
| CERN-OHL-P-2.0 | 宽松开源硬件许可证，适合希望别人复用和商用 |
| CERN-OHL-S-2.0 | 强互惠开源硬件许可证，适合要求衍生硬件继续开源 |
| CC BY-SA 4.0 | 适合文档/图片类资料，但不如 CERN-OHL 适合硬件设计 |

## 建议

本仓库当前策略：

- 硬件设计文件使用 CERN-OHL-P-2.0。
- 文档和图片随硬件资料一起发布。
- 在 README 中说明“硬件资料仅供工程验证，打样和上电需自行承担安全责任”。
