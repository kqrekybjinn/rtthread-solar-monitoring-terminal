# Interfaces

本文件根据当前原理图 PDF 进行接口梳理。正式打样前应以可编辑 EDA 工程和实物丝印再次确认。

## 电源与功率接口

| 接口/网络 | 方向 | 说明 | 注意事项 |
| --- | --- | --- | --- |
| `PV` / `VIN+` | 输入 | 光伏板或外部 DC 输入 | 原理图中 INA226 标注 `VBUS MAX 36V`，输入侧采样耐压需复核 |
| `BAT` / `VBAT` / `VBUS2` | 输出/电池侧 | 接电池、BMS 或后级直流母线 | 必须配合保险丝、BMS 和限流调试 |
| `12V` | 板内辅助电源 | 驱动、风扇和辅助电路供电 | 确认负载能力和隔离策略 |
| `3.3V` | 板内逻辑电源 | MCU、INA226、CAN 等逻辑电路 | 上电前检查无短路 |

## 通信与调试接口

| 接口/信号 | 方向 | 说明 |
| --- | --- | --- |
| `CAN_H` / `CAN_L` | 双向 | CAN 总线接口，经 TCAN3413DR 收发器 |
| `CAN_TX` / `CAN_RX` | MCU 内部 | MCU 与 CAN 收发器连接 |
| `TX` / `RX` | 双向 | 串口调试接口 |
| `SWDIO` / `SWCLK` | 调试 | STM32 SWD 下载调试 |
| `SCL` / `SDA` | 双向 | I2C 总线，连接 INA226 和扩展 I2C 接口 |

## 控制与状态信号

| 信号 | 方向 | 说明 |
| --- | --- | --- |
| `CHG_EN` | 输入/控制 | 充电使能硬线，建议作为硬件级允许信号 |
| `FAULT_N` | 输出/状态 | 故障状态输出，建议低有效或按原理图逻辑确认 |
| `EN` | 控制 | 辅助/驱动使能相关信号 |
| `GPIO16` | 输出 | 原理图中用于风扇/LED 等控制路径之一 |
| `GPIO27` / `GPIO32` / `GPIO33` / `GPIO34` / `GPIO35` | MCU IO | 采样、控制或状态相关，需结合固件引脚表确认 |

| `NTC` | 温度传感器接口 |

## 上电前检查

- 不接光伏和电池，仅给逻辑电源，确认 3.3 V 正常。
- 使用 SWD 确认 STM32G474 可连接、可擦写、可下载。
- 使用限流电源测试辅助 12 V、驱动电源和 3.3 V。
- 不装或禁用功率 MOSFET 时先验证 PWM、驱动 EN、CHG_EN、FAULT_N 逻辑。
- 接电子负载和限流电源验证采样方向、INA226 地址和电流符号。
- 最后再接光伏板和电池/BMS。
