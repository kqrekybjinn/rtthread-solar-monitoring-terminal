# Design Notes

## 设计定位

本仓库发布的是 PCB-02 光伏 MPPT 功率控制节点硬件资料。该节点服务于“基于 Linux + RT-Thread AMP 混合架构的户外光伏自供能智能监测终端”，面向光伏输入、电池充电和直流母线供电场景，配合 STM32G474 + RT-Thread 固件实现采样、MPPT/CC-CV 控制、故障处理和上级 CAN 通信。

该硬件对应竞赛芯片应用赛道中的自主选题方向，重点体现 RT-Thread 在实时采样、功率控制、故障处理、外设管理和模块化扩展中的作用。

## 设计说明

该板的核心设计目标是形成一个可被上级 Linux/RT-Thread AMP 主控管理的 MPPT 功率控制节点：

```text
光伏输入/外部 DC 输入 -> MPPT DC-DC 功率板 -> 电池/BMS/直流母线
```

板上 MCU 运行 RT-Thread，负责本地实时控制和保护；上级系统通过 CAN 下发启停、目标电压、限流、电池类型等参数。功率板本地必须具备安全默认态，不能依赖上级主控才能关断。
